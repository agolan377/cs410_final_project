Magic: The Gathering


This project is a front-end web application that retrieves and displays Magic: The Gathering images and card information using a fan build MTG API.

I developed this application in conjunction with Dan Scott and Forest Pearson for a front-end web development class. The technology stack includes HTML, CSS, Bootstrap, Javascript, React, and a few node.js packages.
Deployed site: https://frontend-project-mtg.herokuapp.com/
Git Repository: https://github.com/scott-dan/frontendProject

Jacuzzi Boys


This project is a full stack web application that retrieves and displays race results for members of the Jacuzzi Boys Athletic Club.

I developed this application in conjunction with Gilbert Grundy for a full stack web development class. The technology stack includes HTML, CSS, and Javascript on the front-end, Node.js and Express on the back-end, and a PostgreSQL database. Unfortunately the site is no longer live.
Git Repository: https://github.com/GilbertGrundy/CS465P-Project
